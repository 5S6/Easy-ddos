# Easy DDOS
Easy DDOS Script :D

## Как установить программу??

MacOS/Linux/Windows

***1.) git clone https://github.com/Alektherblxdev/Easy-ddos***

***2.) cd Easy-ddos***

***3.) python3 hehe.py***


## Как использовать?
`python3 hehe.py -d`

	-h - помощь
	-s - ИП или сайт
	-p - порт - по умолчанию 80
	-t - количество потоков (100-600)
