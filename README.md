# Easy DDOS
Easy DDOS Script

## Как установить программу??

MacOS/Linux/Windows

`git clone https://github.com/Alektherblxdev/Easy-ddos
cd Easy-ddos
python3 hehe.py`

## Как использовать?
`python3 st.py -d`

	-h - помощь
	-s - ИП или сайт
	-p - порт - по умолчанию 80
	-t - количество потоков (100-500)
